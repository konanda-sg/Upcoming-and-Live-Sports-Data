# 🏏 SM Live Sports Data

Automatically fetches, processes and uploads live sports streaming data to JSON using Python + GitHub Actions.

---

## ✨ Features

- 📡 Fetch all live & upcoming events
- 🎬 Auto fetch streams for every event
- 🟢 LIVE / UPCOMING / ENDED status detect
- 📁 Auto generate `Sports_data.json`
- ☁️ Auto upload to GitHub repository
- 🔄 Fully automated using GitHub Actions
- ⚡ Runs every 1hour
